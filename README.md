# Dataset Intelligence Engine (DIE)

Phase 1 of the Universal Dataset Intelligence Platform.

## Core Mission
Build a universal dataset intelligence and ranking platform that understands unknown datasets, discovers structure, entities, relationships, patterns, and anomalies without relying on dataset-specific assumptions.

## Tech Stack
- **Python:** 3.12+
- **Core Processing:** Polars, DuckDB
- **Data Models:** Pydantic v2
- **Graphs:** NetworkX
- **Storage:** Parquet
- **Linting & Formatting:** Ruff
- **Type Checking:** MyPy
- **Testing:** Pytest
